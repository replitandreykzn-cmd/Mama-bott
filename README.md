# 🌸 МамаБот

Telegram-бот помощник для мам. Помогает следить за здоровьем и развитием ребёнка.

## Возможности

- 👶 **Дети** — добавление детей, автоматический календарь прививок
- 📏 **Рост и вес** — история измерений
- 💉 **Прививки** — календарь, отметки о выполненных прививках, напоминания
- 🔔 **Напоминания** — произвольные напоминания на дату и время
- 💊 **Лекарства** — трекер приёма с напоминаниями по расписанию
- 🤒 **Журнал болезней** — температура, симптомы, лечение
- 📄 **PDF-отчёт** — красивый отчёт для педиатра
- 👨‍👩‍👧 **Семейный доступ** — до 5 человек на одну подписку
- ⭐ **Premium** — 14 дней бесплатно, затем 300 ₽/мес

## Запуск на Railway

### Переменные окружения (Environment Variables)

| Переменная | Описание | Пример |
|---|---|---|
| `TELEGRAM_BOT_TOKEN` | Токен бота от @BotFather | `123456:ABC...` |
| `DATABASE_URL` | URL базы данных PostgreSQL (Railway даёт автоматически) | `postgresql://...` |
| `OWNER_TG_ID` | Ваш Telegram ID (для админ-команд) | `123456789` |
| `OWNER_USERNAME` | Ваш username без @ (для кнопки оплаты) | `username` |
| `PREMIUM_PRICE` | Цена подписки (отображается в боте) | `300 ₽/мес` |
| `BOT_TIMEZONE` | Часовой пояс для напоминаний | `Europe/Moscow` |

### Деплой

1. Форкните репозиторий
2. Создайте новый проект на [railway.app](https://railway.app)
3. Подключите GitHub-репозиторий
4. Добавьте сервис PostgreSQL
5. Заполните переменные окружения
6. Railway задеплоит бота автоматически

## Админ-команды

Доступны только владельцу (OWNER_TG_ID):

- `/grant 123456789 30` — выдать Premium на 30 дней
- `/revoke 123456789` — отозвать Premium
- `/stats` — статистика бота
- `/users` — список последних пользователей
- `/broadcast Текст` — рассылка всем пользователям
